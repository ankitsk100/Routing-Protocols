# Verification Commands

Run these on each router:

## Neighbor & Area Checks
- `show ip ospf neighbor`
- `show ip ospf interface brief`
- `show ip ospf`

## Routing Validation
- `show ip route ospf`
- `show ip route`

## Database (optional)
- `show ip ospf database`

## Troubleshooting
- `show run | sec ospf`
- `show ip protocols`
- `ping <remote LAN gateway>`
- `traceroute <remote LAN host>`
